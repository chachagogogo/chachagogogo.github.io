# 자주 쓰는 리눅스 명령어

## 각종 명령어

|                            |                                |
| -------------------------- | ------------------------------ |
| 빈 파일 생성               |  touch 파일명.확장자           |
| 디렉토리 생성              |  mkdir 디렉토리명              |
| 텍스트 파일 확인           |  cat 파일명.txt                |
| 파일, 디렉토리 옮기기      |  mv [file_of_dir] [target_dir] |
| 파일, 디렉토리 이름 바꾸기 |  mv [file_or_dir] [new_name]   |

## 복사

|           |                        |
| --------- | ---------------------- |
| 파일 복사 | cp [file] [target_dir] |
| 폴더 복사 | cd -r [folder]         |

## 삭제

|           |                  |
| --------- | ---------------- |
| 파일 삭제 |  rm [filename]   |
| 폴더 삭제 |  rm -r [dirname] |

# 자주 쓰는 빔 명령어

![](https://pbs.twimg.com/media/Ezq7oi2WQAQV8rr?format=jpg&name=medium)

빔은 insert mode와 ex mode가 따로 있다는 사실을 먼저 숙지하세용~!!!

## ex mode

|     |                                                         |
| --- | ------------------------------------------------------- |
| :w  | 저장하기                                                |
| :q  | 편집기 종료                                             |
| :wq | 저장하고 종료. (파일명까지 입력하면 해당 이름으로 저장) |
| :q! | 저장하지 않고 종료                                      |
